# 🤖 بوت معسكرات الثانوية العامة

بوت تيليجرام متكامل لإدارة معسكرات المذاكرة لطلاب الثانوية العامة، مبني بـ **aiogram 3** و **SQLite** و **Gemini AI**.

---

## ✨ المميزات

- ⛺ **نظام معسكر ذكي** — عداد تنازلي لايف ثانية بثانية مع قفل صارم للجروب
- 👥 **منشن جماعي** — بعد انتهاء المعسكر يُرسَل منشن لكل الأبطال المشاركين
- 🔒 **حذف تلقائي** — أي رسالة من غير المالك تُحذف فوراً أثناء المعسكر
- 🧠 **مستشار Gemini AI** — يرد بالعامية المصرية الحماسية
- 📚 **خطط 30 يوم** — لشعب علمي علوم، علمي رياضة، والأزهر
- 🛠 **لوحة تحكم كاملة** — تعديل كل نص وزرار وإيموجي في البوت بدون لمس الكود

---

## 🚀 التشغيل على Replit (من GitHub)

### 1. استورد المشروع على Replit
- افتح [replit.com](https://replit.com)
- اضغط **Create Repl** ← **Import from GitHub**
- الصق رابط الـ repo

### 2. ثبّت المتطلبات
في Shell:
```bash
pip install -r requirements.txt
```

### 3. اضبط الـ Secrets
في **Tools → Secrets**، أضف:

| الـ Secret | القيمة |
|-----------|--------|
| `BOT_TOKEN` | التوكن من [@BotFather](https://t.me/BotFather) |
| `OWNER_ID` | رقم ID حسابك (من [@userinfobot](https://t.me/userinfobot)) |
| `GEMINI_API_KEY` | مفتاح من [Google AI Studio](https://aistudio.google.com/app/apikey) |

### 4. اضبط الـ Workflow
في **Tools → Workflows**:
```
Command: python bot_v2/bot.py
```
فعّل **Always On** عشان يشتغل 24/7.

## 📋 أوامر البوت

### في المحادثة الخاصة
| الأمر/الزرار | الوظيفة |
|-------------|---------|
| `/start` | فتح القائمة الرئيسية |
| 📚 خطة المذاكرة | خطة 30 يوم للشعب الثلاث |
| 🧠 مستشار الذكاء | الـ AI يرد على أسئلتك |
| `/admin` | لوحة التحكم (للمالك فقط) |

### في الجروب (بعد تعيين البوت أدمن)
| الأمر | الوظيفة |
|-------|---------|
| `/camp 4h` | معسكر 4 ساعات |
| `/camp 30m` | معسكر 30 دقيقة |
| `/camp 1d` | معسكر يوم كامل |
| `/stop` | إيقاف المعسكر يدوياً |

### لوحة التحكم `/admin`
| القسم | الوظيفة |
|-------|---------|
| 📢 إذاعة عامة | رسالة لكل الطلاب |
| 📊 الإحصائيات | عدد الطلاب والإعدادات |
| 🔐 قناة الاشتراك | تعديل الاشتراك الإجباري |
| 📍 جروب المعسكر | تحديد الجروب المعتمد |
| 📝 تعديل الخطط | تعديل خطط الشعب الثلاث |
| ✏️ تعديل محتوى البوت | تعديل كل نص وزرار وإيموجي |
